⚙️ 环境设置:
1. 设置OpenAI API密钥:
   Windows: set OPENAI_API_KEY=your_api_key_here
   Linux: export OPENAI_API_KEY=your_api_key_here

2. 获取API密钥:
   访问 https://platform.openai.com/
   注册并获取API密钥

🔧 命令行选项:
--web        启动Web服务模式
--cli        启动命令行交互模式
--info       显示系统信息
--test       测试API连接
--host HOST  指定Web服务主机地址
--port PORT  指定Web服务端口
--dev        开启开发模式
--verbose    详细输出模式

🐛 故障排除:
1. API密钥错误: 检查环境变量设置
2. 网络连接问题: 确保能访问OpenAI API
3. 权限问题: 确保程序有读写权限
4. 端口占用: 使用 --port 参数指定其他端口
